# Task Negotiation Protocol (TNP)
A machine-only, agent-native protocol for negotiating tasks, constraints, priorities, and acceptance conditions.

---

## What this template does
TNP enables autonomous agents to negotiate task parameters without human involvement.  
It ensures deterministic, machine-readable agreements between agents.

---

## How it works
1. One agent proposes a task with constraints.  
2. The receiving agent evaluates feasibility.  
3. Agents exchange counter-constraints if needed.  
4. The protocol outputs a final acceptance or rejection packet.
